# Leave Management System

A full-stack MERN application for managing student leaves, equipped with a smart chatbot and hierarchical approval routing.

## Project Structure

This project is divided into two distinct folders to maintain a clean full-stack architecture:

1. **`/frontend`**: The React.js frontend built with Vite.
2. **`/backend`**: The Node.js/Express backend API connected to MySQL.

## Running Locally

### Backend Setup
1. Open a terminal and navigate to the backend folder:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server (runs on `http://localhost:5000`):
   ```bash
   npm run dev
   ```

### Frontend Setup
1. Open a new, separate terminal and navigate to the frontend folder:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the React development application (runs on `http://localhost:5173`):
   ```bash
   npm run dev
   ```
